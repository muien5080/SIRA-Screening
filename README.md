# SIRA Screening - GSoC 2026

## Overview
This repository contains the **SIRA (Susceptible-Infected-Removed Approximation) screening project** for CERN’s Human-AI initiative.  

The main goals are:  
1. Simulate epidemic data using the **stochastic SIR model**.  
2. Train **machine learning models** to predict mean counts of `S(t)`, `I(t)`, `R(t)`.  
3. Use **symbolic ML methods** to approximate the deterministic SIR equations.

This project serves as a **screening exercise** for the upcoming GSoC 2026 application.

---

## AI Disclosure

This project leverages machine learning models to predict epidemic dynamics using the SIR model. Specifically, we are training models using synthetic epidemic data to predict the mean values of S(t), I(t), and R(t) over time.

### The AI techniques used in this project are:
Supervised learning to approximate epidemic models.
Symbolic ML methods (using auto-differentiation) to approximate deterministic SIR equations.

Please note that the models trained in this repository are based on synthetic epidemic data generated by the SIR simulations, and are intended to approximate epidemic dynamics rather than predict real-world outbreaks
